# NADIA Agent

> Autonomous AI Agent Platform for Android, Linux, and Cloud.

## Overview

NADIA Agent is an open-source autonomous AI framework designed to think, plan, execute, and improve tasks independently.

Unlike a traditional chatbot, NADIA Agent is built as an extensible agent platform capable of using tools, managing workflows, and collaborating with multiple AI providers.

The project follows a **Mobile First** philosophy, allowing development directly from Android using Termux while remaining fully compatible with Linux, VPS, and cloud environments.
